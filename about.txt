🚀 Hosting Instructions (GitHub Pages)

To host this portfolio on your custom domain using GitHub:

Create a Repository: Create a new public repository on GitHub named portfolio.

Upload Files: Upload index.html, README.md, and your image file TanishaPic.jpg to the root directory.

Enable Pages: Go to Settings > Pages. Set the source to "Deploy from a branch" and the branch to main.

Custom Domain: Enter tanishasmith.cloud in the Custom Domain field. Configure your AWS Route 53 settings with GitHub's IP addresses (A records) and a CNAME record for the www subdomain.
